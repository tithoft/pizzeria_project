# pizzeria_project

Pizzeria Project

Pizzeria Project is a simple Django web application that displays a list of pizzas and the toppings associated with each one.
It is designed as an educational project to demonstrate the structure and mapping of a basic Django application—showing how models, URLs, views, and templates interact within a Django project.

Features

View a list of all available pizzas

Click on a pizza to view its detailed toppings

Simple, clean Django project layout for learning purposes

Uses Django’s ORM to map relational data (Pizza → Toppings)
